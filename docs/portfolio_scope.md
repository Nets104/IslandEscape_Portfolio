# Portfolio Scope

이 저장소의 기준 문서: `Portfolio_KimSeohyun_IslandEscape.pdf`

## PDF 기능 범위

- 월드 아이템, 하이라이트, 픽업/드롭
- 인터랙션 인터페이스와 조준 보정
- 인벤토리/퀵슬롯 분리 구조
- `FItemInstance` 기반 수량/내구도 이동
- 채집 시스템
- 제작대/모닥불 UI와 DataTable 기반 레시피
- 트러블슈팅 1: 아이템 복제/삭제 방지
- 트러블슈팅 2: 물병 소비 판정 분리
- 배 수리/탈출 UI 흐름

## 정리 기준

PDF에서 설명한 기능을 기준으로 소스 파일을 기능별 카테고리로 분류했다.
