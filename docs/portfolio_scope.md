# Portfolio Scope

이 저장소의 기준 문서: `Portfolio_KimSeohyun_IslandEscape.pdf`

## PDF에 직접 등장하는 기능

- 월드 아이템, 하이라이트, 픽업/드롭
- 인터랙션 인터페이스와 조준 보정
- 인벤토리/퀵슬롯 분리 구조
- `FItemInstance` 기반 수량/내구도 이동
- 채집 시스템
- 제작대/모닥불 UI와 DataTable 기반 레시피
- 트러블슈팅 1: 아이템 복제/삭제 방지
- 트러블슈팅 2: 물병 소비 판정 분리

## 제외 기준

PDF에서 기술 설명 대상으로 다루지 않은 AI, 몬스터, 보스 전투, 닭/호랑이, 스포너, 맵 에셋은 공개용 소스에서 제외한다.

전체 프로젝트가 아니라 포트폴리오 증빙용 선별본이므로, 컴파일 가능한 완성 프로젝트 구성을 목표로 하지 않는다.
